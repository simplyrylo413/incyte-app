NEXT UP — 3 treatment options

Three alternatives to the current floating "NEXT" pill + left-edge accent on the Today screen's first uncompleted movement. Same card content in each so the treatment compares directly.

A

Ambient glow

No chip, no edge bar. The card simply lifts off the page with a soft accent-blue ambient shadow. Whisper-emphasis: you see it before you can name what's different.

Character Most subtle. No language, no chrome. The card just feels alive. Tradeoff: a returning user might miss it on a quick glance; ambient cues require attention. Best when the user is already in flow and doesn't need an instruction.
B

START CTA chip

Repurpose the existing progress chip into a primary action. "0/4" becomes a solid accent-blue "Start →" pill on the next-up card. The chip is the cue. Tap it to enter.

Character Most action-oriented. The cue and the action are the same element — tap "Start" to enter. Tradeoff: it shifts the chip's meaning per-card (progress vs. CTA). After you log a set on Bench Press, the chip flips back to "1/4" progress and the next card's chip becomes "Start". The user adapts to that — call it action-first design.
C

Inline marker

No card-level chrome at all. A small accent dot + "NEXT" mono label sits inline, immediately before the movement name. Reads as part of the row, not a stuck-on decoration.

Character Most editorial. Treats "NEXT" as content rather than UI chrome. The dot + small label live inline with the name, sharing its baseline. Card geometry stays identical to non-next cards — no shadow swap, no border change. Tradeoff: less visual punch than A or B; relies on the user noticing the inline glyph rather than feeling the card.
